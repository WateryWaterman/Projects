# 项目说明

- Readme.txt 是这个产品的初步构想，也是喂给 AI 的提示词，各个 AI 给出的结果链接在txt文末。
- 详细版策划案见 outline final.html（outline final.ipynb 更易读，但需用 Python 打开）。
- AI声明：最终策划案和视频借助 AI 完成了包括结构生成、图片视频生成、配乐生成等环节。
- html 文件的原始 ipynb 与最终成片已上传至谷歌硬盘，链接位于 Silic Diary.html 中。

## 整体工作流

想法 → 模仿产品逻辑撰写策划案及 prompt → 给 AI 写好具体任务并比较、综合不同 AI 的结果 → 审核得到最终策划案（做视频最重要的是产品概念与视频时间分配）→ 真人视频录制 → 豆包 AI 生成解说图片和概念图标 / Figma 生成产品效果 → Pr 剪辑 → SunoAI 生成配乐。
